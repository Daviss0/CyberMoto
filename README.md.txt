# CyberMoto

Sistema de e-commerce voltado para a venda de peças e acessórios de motos. Projeto desenvolvido com Java, Spring Boot, Thymeleaf e H2.

## Funcionalidades atuais

- Autenticação de usuários com Spring Security
- Login com controle de acesso por tipo de usuário (`ADMIN`, `ESTOQUISTA`)
- Cadastro de usuários com senha criptografada (BCrypt)
- Integração com banco H2

## Tecnologias usadas

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- H2 Database
- Git + GitHub

## Requisitos para rodar

- JDK 17+
- Maven 3.8+
- IntelliJ ou outro IDE com suporte a Spring Boot
